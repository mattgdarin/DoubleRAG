Machine learning is a subset of artificial intelligence where systems learn from data to improve performance on tasks without being explicitly programmed.

There are three main types of machine learning: supervised learning, unsupervised learning, and reinforcement learning.

In supervised learning, models are trained on labeled data — input-output pairs — and learn to map inputs to outputs.
Common supervised tasks include classification and regression.

Unsupervised learning finds structure in unlabeled data.
Clustering and dimensionality reduction are typical unsupervised tasks.

Reinforcement learning trains agents to take actions in an environment to maximize a cumulative reward signal.
